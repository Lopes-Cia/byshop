## Padrão de mensagem de commit

- Use Conventional Commits: `type(scope): mensagem curta`.
- Tipos preferidos: `feat`, `fix`, `refactor`, `chore`, `docs`, `test`.
- Mensagem curta em português, no imperativo (ex.: "ajusta validação do login").
- Se houver breaking change, use `!` e descreva no corpo.
